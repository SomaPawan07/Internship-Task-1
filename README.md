# Data Cleaning Task

This project demonstrates how to clean and prepare a raw dataset using Python (pandas).

## Steps Performed
1. Handled missing values in the `Income` column.
2. Removed duplicate records.
3. Converted `Dt_Customer` column to datetime format.
4. Standardized text fields (`Education`, `Marital_Status`).
5. Removed outliers using the IQR method.

## Files Included
- `data_cleaning.ipynb` — Code used for cleaning.
- `marketing_campaign.csv` — Original dataset.
- `cleaned_data.xlsx` — Cleaned output dataset.

## Tools Used
- Python 3
- pandas
- xlsx
